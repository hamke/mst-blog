KakaoPay 受取用QRコードの確認と画像のダウンロード方法

2025-05-06 / KakaoPay  
更新日 : --01

MoneySkillTalkでは、KakaoPayの受取用QRコードを確認し、画像をダウンロードする方法をご紹介します。

まず、KakaoTalkのモバイルアプリを開き、「More」メニューの上部にある「Scan」ボタンをタップします。

「Scan」画面が表示されたら、画面下部の「Receive」メニューをタップします。

「Receive」をタップすると、自分専用の KakaoPay 受取用QRコードが表示されます。

「金額入力」は任意です。金額を入力すれば、その金額専用のQRコードが生成され、未入力の場合は送金者が自由に金額を入力できるQRコードが自動生成されます。

QRコードの下にあるダウンロードアイコンをタップすれば、生成されたQRコード画像を保存できます。

KakaoPay モバイル送金リンク生成サービス(KakaoPay Mobile Transfer Link Generator)は、ダウンロードした受取用QRコードから簡単に送金リンクを作成できるオンラインサービスです。

生成されたリンクをWebサイトやブログに設置すれば、面倒な手続きなしで、クリックひとつでKakaoPayアカウントへ送金を受け取ることができます。

KakaoPay モバイル送金リンク生成サービス (KakaoPay Mobile Transfer Link Generator) : 詳細はこちら
www.mkttalk.com/tools/kakaopay-link-generator/
